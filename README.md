# Coreum Central - ECG WINCARDIO Alerts

Userscript para **Tampermonkey** que monitora a fila da página `Coreum Central` e dispara alertas sonoros + notificações nativas quando surgem exames **ECG WINCARDIO**, quando o **SLA está próximo de vencer** e quando o **SLA já está vencido**.

---

## Objetivo

Esse script foi criado para ajudar médicos a identificarem rapidamente exames novos na fila, reduzindo atraso de atendimento e evitando estouro de SLA.

Página monitorada:

```text
https://classic.coreum.health/classic/central
